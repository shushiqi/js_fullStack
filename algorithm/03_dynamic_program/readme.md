# 动态规划
## 老虎吃羊
    500只虎， 1只羊
    1. 都吃草
    2. 老虎可以吃羊， 一只虎吃一只羊，吃完羊后这只老虎变成羊，要不要吃掉这只羊？

找到规律后 可以递归 或类推  
1. 吃什么羊,虎变羊，被欺
2. 吃， 羊肉好吃
3. 不吃 羊很可爱

一只老虎    一只羊  吃=>不会被其他老虎吃掉  
两只老虎    一只羊  不吃=>会被其他老虎吃掉     
三只老虎    一只羊  吃=>不会被其他老虎吃掉    
四只老虎    一只羊  不吃    
五只老虎    一只羊  吃  

奇数就吃 偶数不吃

## 动态规划： 大事化小，小事化了
    最优子结构：上一次不吃，下一次就吃  
    边界 n = 1吃 n = 2不吃 =>递归   
    状态转移 F(n) = !F(n-1)


### 楼梯只有12阶，一步走1阶或2阶 请问走完楼梯有多少种走法？
    1. 边界n=1时    1种 走1步   
       边界n=2时    2种
    2. 子结构 n=12时  有多少种走法
    n=11时 n=10时
    状态转移 F(n) = F(n-1) + F(n-2)
